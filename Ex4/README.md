# image_processing

Description. 
The package processamento_imagem is used to:
	Processing:
		- Histogram matching
		- Structural similarity
		- Resize image
	Utilidades:
		- Read image
		- Save image
		- Plot image
		- Plot results
		- Plot histogram

## Installation

Use the package manager [pip](https://pip.pypa.io/en/stable/) to install Image_processing

```bash
pip install Image_processing
```
